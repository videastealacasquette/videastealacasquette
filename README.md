
# Vidéaste à la Casquette — Site Next.js + Tailwind (App Router)

## Lancer en local
```bash
npm install
npm run dev
# ouvre http://localhost:3000
```

## Déployer sur Vercel
- Crée un compte sur Vercel, connecte ton repo (ou import ZIP)
- `npm run build` localement n'est pas nécessaire, Vercel s'en charge
- Attache ton domaine `videastealacasquette.fr` (onglet Domains)

## Où modifier ?
- `app/page.tsx` : contenu principal (texte, sections, liens)
- `public/Logo_VALC_Couleurs.png` : ton logo
- `app/layout.tsx` : SEO de base
